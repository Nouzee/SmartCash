# Name the project SmartCash

The independent directional high-frequency microstructure project is named **SmartCash**. SmartCash owns its end-to-end strategy product and may use Lemnis as a high-frequency infrastructure library; “smart-money” remains only the temporary repository and Python package name until a controlled migration updates paths, imports, artifacts, and documentation together.

## Consequences

- New domain documentation and user-facing research reports use SmartCash as the only project name.
- Repository and package renaming is an implementation migration, not an ad-hoc filesystem move during specification grilling.
- Compatibility shims or artifact-path migration must be explicit when the rename is implemented.
