# Name the project SmartCash

The independent directional high-frequency microstructure project is named **SmartCash**. SmartCash owns its end-to-end strategy product and may use Lemnis as a high-frequency infrastructure library. The controlled migration updates the repository, Python package, commands, imports, artifacts, and documentation together while retaining only explicit compatibility aliases where needed.

## Consequences

- New domain documentation and user-facing research reports use SmartCash as the only project name.
- Repository and package naming use `smartcash`; the legacy `smart_money` package path is not a second public API.
- Compatibility shims or artifact-path migration must be explicit when the rename is implemented.
